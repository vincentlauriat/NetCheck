features

Four tools, one little globe

Built on strict Swift 6 concurrency with Apple's own network engine. No sandbox theatre — just accurate, real-time answers.

🌐

Connectivity globe

A green / orange / red globe icon, always visible in the menu bar, reflecting your live connection state.

📡

WiFi Finder

Walk around with a Geiger-counter sound and concentric waves to home in on the strongest spot.

⚡

Speed Test

Download, upload, RPM and latency measured with Apple's networkQuality engine.

🗺️

Traceroute

An animated 3D globe — the camera flies from space to each hop, then zooms back out.

📊

Usage Quality

A single latency measurement mapped to hierarchical thresholds, from mail to gaming.

⚙️

Preferences

Launch at login via SMAppService, with Sparkle auto-updates.

get started

Up and running in 30 seconds

Signed, notarized and Hardened-Runtime — it opens without any Gatekeeper warning.

Download

Grab the latest .dmg from GitHub Releases and mount it.

Install

Drag NetCheck.app into /Applications.

Launch

Open it — the globe lands in your menu bar. Requires macOS 26 or later.
